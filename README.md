# babel-minify

Some babel plugins and presets to minify.

## babel-preset-min

The plugins included in the preset - https://github.com/boopathi/babel-minify/blob/master/packages/babel-preset-min/index.js

#### .babelrc

```json
{
  "presets": ["min"],
  "comments": false,
  "compact": true,
  "minified": true
}
```
